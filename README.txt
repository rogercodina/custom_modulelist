Custom Module list
==================

Creates a block and a page with plain text module list with no anable/disable/configure/help links for desired roles.

This module:

1) Adds a permission called "See modules list" which you have to assign to roles you wish can see plain text module list
2) Creates a block named "Custom module list" with the plain text module list
3) Creates a page in admin/modulelist with the plain text module list
